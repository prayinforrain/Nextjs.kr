title: app 디렉토리
description: App 라우터가 레이아웃, 스트리밍 등을 사용할 수 있게 합니다.
---

> **참고**: Next.js 13.4부터는 이 옵션이 **더 이상** 필요하지 않습니다. App 라우터는 안정화되었습니다.

App 라우터는([`app` 디렉토리](/docs/app/building-your-application/routing)) [레이아웃](/docs/app/building-your-application/routing/pages-and-layouts), [서버 컴포넌트](/docs/getting-started/react-essentials#server-components), [스트리밍](/docs/app/building-your-application/routing/loading-ui-and-streaming), 그리고 [colocated data fetching](/docs/app/building-your-application/data-fetching)을 지원합니다.

`app` 디렉토리를 사용하면 [React Strict 모드](https://react.dev/reference/react/StrictMode)가 자동으로 활성화됩니다. [`app`을 점진적으로 적용하는](/docs/pages/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)하는 법을 배워보세요.
